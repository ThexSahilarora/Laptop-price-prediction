# 💻 Laptop Price Prediction

## 📌 Problem Statement
The goal of this project is to predict laptop prices based on various features like brand, RAM, processor, storage, etc.

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- GridSearchCV

## ⚙️ Machine Learning Process
1. Data Cleaning
2. Encoding Categorical Variables
3. Feature Scaling
4. Train-Test Split
5. Model Building
6. Hyperparameter Tuning using GridSearchCV
7. Model Evaluation

## 🤖 Models Used
- Random Forest Regressor
- XGBoost Regressor

## 📊 Best Model Accuracy
R2 Score: (0.9995510173393443)

## 🚀 How to Run
1. Install required libraries
2. Run the Jupyter notebook
3. Train the model
